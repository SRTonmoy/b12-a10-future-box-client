# HabitHub (Client)

A Habit Tracker single-page app built with React, Firebase Auth, Tailwind and an Express backend.

## Live site
(https://future-box-api-client.web.app/)

## Features
- Email / Password and Google Authentication (Firebase)
- Add, Update, Delete personal habits (private routes)
- Browse latest public habits (6 newest on home)
- Mark habit complete with streak logic and completion history
- Search and filter public habits, responsive UI, toast & sweet alerts
- Animated home sections with Framer Motion

## Run locally
1. Copy `.env.example` to `.env` and fill values.
2. `npm install`
3. `npm run dev`
